◆ get()

template<typename ValueType >
RFK_NODISCARD ValueType rfk::Variable::get ( ) const

Get the value stored in this variable. This method in not safe if you provide a wrong ValueType.

Template Parameters
ValueTypeType of the value stored in the field. If ValueType is an rvalue reference, the value is moved into the return value (so the variable is no longer safe to get). If ValueType is an lvalue reference, return a reference to the variable. If ValueType is a value type, the value is copied. If it is a struct/class, ValueType must be copy-constructible.
Returns
The value stored in the variable.
Exceptions
ConstViolationif:
  • the field is const and ValueType is an RValue type (can't move a const field content);
  • the field is const and ValueType is a non-const reference;
