◆ get()

template<typename ValueType >
RFK_NODISCARD ValueType rfk::StaticField::get ( ) const

Get the data corresponding to this static field. This method in not safe if you provide a wrong ValueType.

Template Parameters
ValueTypeType to retrieve from the field. If ValueType is an rvalue reference, the data is moved into the return value (so the class data is no longer safe to use). If ValueType is an lvalue reference, return a reference to the field. If ValueType is a value type, the data is copied. If it is a class, ValueType must be copy-constructible.
Returns
The queried data.
Exceptions
ConstViolationif:
  • the static field is const and ValueType is an RValue type (can't move a const field content);
  • the static field is const and ValueType is a non-const reference;
